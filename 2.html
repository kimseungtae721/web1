CSS

CSS란 웹 문서의 전반적인 스타일을 미리 지정해둔 '스타일시트'입니다. 문서 전체의 일관성을 유지할 수 있고, 세세한 스타일 지정의 필요를 줄어들게 합니다. [출처 - 두산백과] 앞서 말씀드린 HTML로 웹 페이지를 만들고, CSS로 그것을 꾸며줄 수 있습니다. 예를 들어, 제목은 글자 크기도 크고 두껍게 표시하고 내용은 작은 글자 크기에 얇게 표시하고 싶다면! HTML로 제목과 내용을 작성하고 CSS로 제목과 내용에 대한 스타일을 꾸밀 수 있답니다 :) 웹 페이지를 쭉 만들다가 또 제목과 내용이 들어가야 한다면? 간단한 태그만 사용한다면 앞서 지정한 CSS 스타일이 그대로 입혀지게 된답니다. 그렇게 되면 문서 전체의 일관성을 유지할 수도 있고 작업 시간도 훨씬 단축되겠죠?
